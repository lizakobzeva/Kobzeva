# Лабораторная работа 1: Сериализация JSON, управление ресурсами и работа с файлами в C#

## Цели работы
1. Освоить использование атрибутов JSON для управления процессом сериализации/десериализации.
2. Изучить паттерн IDisposable для ĸорреĸтного освобождения ресурсов.
3. Научиться работать с файловой системой (чтение/запись файлов).
4. Реализовать сохранение и загрузĸу объеĸтов в/из файлов JSON.
5. Понимать разницу между явным освобождением ресурсов через Dispose() и автоматичесĸим через
   финализатор.

## Требования
- .NET 8.0 или выше
- Нет внешних зависимостей, кроме System.Text.Json, System.IO и xUnit

## Классы
- **Person**: Представляет человека с кастомизированными свойствами JSON (например, игнорируемый пароль, переименованные поля).
- **PersonSerializer**: Обеспечивает сериализацию, десериализацию и сохранение/загрузку файлов (синхронно/асинхронно) для одиночных объектов и списков.
- **FileResourceManager**: Управляет потоками файлов с IDisposable, поддерживая операции чтения/записи/добавления.
- **PersonSerializerTests**: Unit-тестирует функциональность класса PersonSerializer.

## Примечания
- Исключения логируются в "error.log".
- Вывод JSON отформатирован для читаемости.
- Потокобезопасность обеспечена через lock.

## Сборка и запуск
- Скомпилируйте проект с помощью dotnet build, а затем запустите исполняемый файл для использования.
- Запустите тесты с помощью dotnet test.

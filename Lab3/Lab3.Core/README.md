# Лабораторная работа 3: Реализация коллекций и интерфейсов

## Цель работы
Познаĸомиться с интерфейсами ĸоллеĸций в C# (.NET), реализовать простые ĸоллеĸции 
и понять их ĸонтраĸт и фунĸциональность.

## Требования
- .NET 8.0 или выше
- Нет внешних зависимостей, кроме System.Collections и xUnit

## Классы
- **SimpleList**: Реализует интерфейсы (все методы и свойства) IEnumerable, ICollection, IList, 
внутренне использует динамичесĸий массив.
- **DoublyLinkedList**: Реализует интерфейсы (все методы и свойства) IEnumerable, ICollection, IList,
внутренне использует двухсторонний связный список (используются классы Node с полями Value, Previuous и Next).
- **SimpleDictionary**: Реализует интерфейс IDictionary<TKey, TValue>, 
внутренне использует список из списков пар ключ-значение. Также реализует интерфейс IReadOnlyDictionary.
Был выбран именно этот интерфейс вместо указанных IReadOnlyCollection / IReadOnlyList, так как он предоставляет
все возможности IReadOnlyCollection (расширяет его) и аналогичные IReadOnlyList расширяющие возможности
(IReadOnlyList также расширяет IReadOnlyCollection), только для словарей:
доступ не по индексу, а по ключу, вместо гарантированного порядка - уникальность элементов.

## Сборка и запуск
- Скомпилируйте проект с помощью dotnet build, а затем запустите исполняемый файл для использования.
- Запустите тесты с помощью dotnet test.
